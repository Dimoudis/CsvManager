This app is part of my master’s degree project at the university
It focuses on managing data from csv files 

◉ Current features

initial commit:

- csv file upload

- the table is dynamically created with headers and rows

- tt supports pagination for large files (10 rows per page)

- each column has a checkbox in a dropdown

- show / hide columns: users can show or hide columns

- page navigation: “Prev” and “Next” buttons allow users to navigate between pages

second commit:

- add form for new records

- add validation on record creation (not empty fields)

- fix pagination (from first page you can go to the last (prev), and from the last page you can go to the first (next))

third commit:

- add delete record with delete confirmation dialog

fourth commit:

- undo delete (only for last deleted record)
